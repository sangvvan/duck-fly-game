# Duck Fly Game - iOS SwiftUI 2D Mini Game

A simple 2D iOS mini game where a duck flies around to collect falling food items.

## Game Overview
- **Platform**: iOS (SwiftUI)
- **Genre**: 2D Casual Game
- **Objective**: Move the duck to catch falling food and earn points

## Current Features
- ✅ Duck character (emoji-based)
- ✅ Falling food items (corn emoji)
- ✅ Mouse/touch tracking to move duck
- ✅ Collision detection
- ✅ Score system
- ✅ Game loop with display link for smooth 60fps

## How to Run
1. Open in Xcode (SwiftUI requires iOS 15+)
2. Select an iOS simulator or device
3. Run the game
4. Move your cursor/touch to control the duck
5. Catch the falling food to earn points

## Game Controls
- **Mouse/Touch**: Move duck to catch food
- The duck follows your cursor position
